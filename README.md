Basic CSS Syntax
A CSS rule consists of a selector and a declaration block: 
css
selector {
  property: value;
  property: value;
}
selector: Points to the HTML element(s) you want to style (e.g., body, h1, p).
property: The attribute you want to change (e.g., color, font-size, background-color).
value: The new value for the property (e.g., blue, 20px, lightblue).
: (colon): Separates the property from the value.
; (semicolon): Separates different declarations within a block.
